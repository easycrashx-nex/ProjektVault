Arcane Vault Online Vorbereitung

Dein Kartenkonto wartet schon.

Diese Version speichert alles lokal, ist aber bereits so aufgebaut, dass sie später auf einen Server umgestellt werden kann. Erstelle jetzt ein Konto und starte mit fünf kostenlosen Starter-Boostern.

  • Mehr als 200 Karten inklusive Trainerkarten
  • Mehrere Booster-Stufen mit unterschiedlichen Chancen
  • Gespeicherte Decks mit Verfügbarkeitsprüfung

Speicherort: lokal im Browser. Später kann das auf Server-Login umgestellt werden.
